left_locations = left_merge.location.unique()

import numpy as np
is_present = np.isin(left_merge.location.unique(),right_merge.location.unique())

is_presnt is a boolean array that tells us if a given element of left_locations is present in right_locations. We want to extract the False values, so we need to do indexing with a “negated” version of is_present:
